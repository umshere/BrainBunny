# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-07-26

### Added
-   **Initial Release!**
-   **Guest Mode:** Full application functionality without user accounts. Session data is saved locally.
-   **Multi-Student Profiles:** Parents can create and manage multiple student profiles.
-   **AI Worksheet Generator:**
    -   Two-step process: Generate questions, then review and finalize.
    -   Generate questions for various subjects, topics, and grade levels.
    -   Format questions for printing or assign as an interactive quiz.
-   **Interactive Student Zone:**
    -   Students can complete assigned quizzes on the platform.
    -   Dynamic quiz card supports Multiple Choice, Fill-in-the-Blank, and Short Answer questions.
    -   Gamified feedback for correct/incorrect answers.
-   **Adaptive Learning:** "Weak Points" system tracks topics from incorrect answers for targeted practice.
-   **Worksheet Library:** AI-generated topic ideas to inspire parents.
-   **Accessibility:** Font size controls for quiz questions.

### Fixed
-   **Mobile Responsiveness:** Improved layout and prevented scrolling issues on the landing page and quiz screen for smaller devices.
-   **UI Alignment:** Corrected text alignment in the quiz feedback/explanation box.
