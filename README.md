# BrainyBunny.ai - AI Homework Helper

BrainyBunny.ai is a powerful and intuitive application designed to help parents and teachers generate high-quality, printable homework questions for children. Using advanced AI, it can create worksheets from a simple topic, or even analyze a student's completed work to generate targeted practice for their specific weak points.

## Core Features

- **Generate from Topic:** Create worksheets on any subject (e.g., "The Water Cycle," "Simple Addition").
- **Target Weak Points:** Upload an image of a completed worksheet, and the AI will identify mistakes and create a new sheet to practice those specific skills.
- **Interactive Student Zone:** A fun, game-like quiz mode for kids to test their knowledge. Features include difficulty levels, a "level-up" progression system, and lives. Incorrect answers are saved as "weak spots" for parents to create targeted practice worksheets.
- **Customizable Worksheets:**
    - Select Grade Levels (Kindergarten - 8th Grade)
    - Specify the exact Number of Questions
    - Choose Question Types (Multiple Choice, Word Problems, Fill-in-the-Blank, or Mixed)
- **Automatic Answer Key:** Every worksheet is generated with a corresponding answer key, often including step-by-step solutions.
- **Advanced Print Preview:** A fully responsive print preview allows you to adjust page size, margins, font size, and question spacing to get the perfect printout on any device.
- **Multimedia Input:** Supports text input, voice input via microphone, and image uploads from a file or directly from the device camera.

## Product Roadmap

This roadmap outlines the future vision for BrainyBunny.ai, transforming it into a comprehensive and personalized learning platform.

### Phase 1: Immediate Enhancements (Core AI & UI)

*These features can be implemented with the current architecture and focus on expanding content variety and user choice.*

- **Worksheet Themes & Aesthetics:**
    - **[ ] Minimal Clean:** A simple, elegant design.
    - **[ ] Ink Saver:** A theme with no heavy lines or filled-in colors, optimized for printing.
    - **[ ] Cute & Illustrated:** A theme with fun, kid-friendly illustrations and borders (AI-generated).
    - **[ ] Seasonal/Festival Themes:** Special packs for holidays like Christmas, Diwali, Onam, Spring, etc.
- **Enhanced Export Options:**
    - **[ ] DOCX Export:** Allow teachers to download worksheets as editable Microsoft Word documents.
    - **[ ] Image Export (PNG/JPG):** Enable easy sharing and use on digital platforms like tablets.
- **Worksheet Library & Packs:**
    - **[ ] Curated Skill Packs:** Create prompts for "starter packs" on common topics (e.g., "Fractions Starter Pack," "Subtraction with Regrouping Pack").
    - **[ ] Browse by Skill:** Allow users to browse and generate worksheets from a pre-defined list of common skills by grade.

### Phase 2: Intelligence & Personalization

*This phase focuses on leveraging AI to create a more adaptive and insightful learning experience.*

- **Smart Weak-Point Detection:**
    - **[ ] Deep Error Analysis:** When a worksheet is uploaded, train the AI to detect specific error patterns:
        - *Math:* Borrowing/carrying mistakes, operation confusion (+ vs ×), place-value misunderstandings.
        - *Language:* Spelling vs. grammar errors, reading comprehension issues.
    - **[ ] Automated Targeted Practice:** Automatically generate a worksheet that directly addresses the detected error patterns.
- **Single Question Regeneration:**
    - **[ ] "Try Another One":** Add a button next to each question on a generated worksheet to regenerate a new, similar question of the same type.
- **Parent Coaching Tips:**
    - **[ ] Actionable Advice:** Alongside a "weak points" analysis, generate a coaching tip for the parent, e.g., *“It looks like Leo is struggling with regrouping. Try using base-ten blocks to show the concept visually.”*

### Phase 3: User Accounts & Progress Tracking

*This phase requires a backend and database to store user data and enable long-term tracking.*

- **User Modes & Profiles:**
    - **[x] Guest Mode:** The current default, no history saved.
    - **[x] Parent/Teacher Mode:** Manage profiles for one or more students.
    - **[x] Student Mode:** A simplified, motivational UI for the child.
    - **[ ] Student Avatars & Profiles:** Let kids choose a fun avatar (bunny, robot, dinosaur) for their profile.
- **Progress Tracking Dashboard:**
    - **[ ] Worksheet History:** View all past worksheets by date and topic.
    - **[ ] Skill Mastery Chart:** Visualize a student's progress in different skills (e.g., a simple bar chart showing mastery in "Addition," "Subtraction," etc.).
    - **[ ] Weekly Summary:** Generate a simple "Weekly Improvement" card for parents.
- **Adaptive Difficulty:**
    - **[ ] Smart Progression:** Automatically adjust the difficulty of new worksheets based on a student's past performance.

### Phase 4: Community & Classroom Features

*The long-term vision to make BrainyBunny a collaborative tool for educators and parents.*

- **Motivation & Reward Layer (Gamification):**
    - **[x] XP/Star System:** Award points for completing worksheets.
    - **[ ] Daily Streaks:** Encourage consistent practice with a streak badge.
    - **[ ] Unlockable Content:** Use points to unlock new worksheet themes or avatars.
    - **[ ] Fun Finishers:** Add a "Great Job!" mini-coloring doodle at the bottom of a completed worksheet.
- **Classroom Dashboard:**
    - **[ ] Manage Multiple Students:** A dedicated view for teachers to manage their entire class.
    - **[ ] Assign Worksheets:** Assign a specific worksheet to one or more students.
    - **[ ] Export to Google Classroom:** Seamlessly integrate with existing classroom tools.
    - **[ ] Shareable Links:** Generate a unique link to a worksheet for easy distribution.
